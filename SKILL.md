# Tax Agent SKILL — 세무 지식 참조

소득세법 각 챕터에서 Python 계산 함수 없이 텍스트 안내가 필요한 항목.
사용자 질문에 답변할 때 이 내용을 참조한다.

---

## Ch01. 소득세 총설

### 납세지 규정 (소득세법 §6~§8)

납세지 = 세무신고·납부 관할 세무서를 결정하는 기준지.

| 납세자 유형 | 납세지 원칙 |
|---|---|
| 거주자 | 주소지 (주소 없으면 거소지) |
| 비거주자 | 국내원천소득 발생지 (부동산: 소재지, 사업: 사업장) |
| 사업소득자 — 사업장 선택 시 | 사업장 소재지 (§8 납세지신고서 제출 필요) |

- **납세지 변경**: 주소 이전 시 자동 변경, 별도 신고 불요
- **사업장 납세지신고**: 사업소득자가 주소지 대신 사업장을 납세지로 원하면 사업장 관할 세무서에 신고서 제출 (§8①)
- **공동사업장**: 대표 공동사업자 주소지 또는 사업장 소재지

## Ch02. 이자소득

### 이자소득 수입시기 (소득세법 §45①1)

수입시기 = 이자소득이 귀속되는 과세기간 결정 기준.

| 이자 유형 | 수입시기 |
|---|---|
| 약정에 의해 지급일이 정해진 이자 | **약정일** |
| 약정일 이전에 실제 지급된 이자 | **지급일** (앞당겨진 날) |
| 채권·증권 만기 이자 | 만기일 (중도상환 시 상환일) |
| 보통예금·수시입출금 이자 | 실제 통장 입금일 |
| 원본에 전입되는 이자 | 원본 전입일 |

**실무 포인트**: 12월 만기 정기예금 이자를 1월에 수령해도 → 만기일(12월)이 수입시기. 해당 과세연도 귀속.

## Ch03. 배당소득

### 집합투자기구 배당소득 특례 (소득세법 §17①5)

집합투자기구(펀드·ETF 등)에서 받는 이익분배금의 과세 처리.

| 구분 | 과세 여부 |
|---|---|
| 환매(해지) 시 수익: 환매금액 - 취득가액 | 배당소득 과세 (원천징수 14%) |
| 펀드 내 국내주식 매매익 | **비과세** (§94①3 양도소득 과세 대상 아님) |
| 펀드 내 채권 이자 | 이자소득으로 구분 과세 |
| 분배금(결산 지급) | 배당소득 과세 |
| 해외주식형 ETF(국내 상장) — 주식매매익 분배 | 배당소득 과세 (해외주식 양도익은 비과세 아님) |

- **Gross-up 불적용**: 집합투자기구 배당소득에는 배당세액공제(Gross-up) 적용 안 됨 (§17③ 단서)
- **손실 환매**: 환매금액 < 취득가액이면 배당소득 0원 (음수 불인정)

## Ch04. 사업소득

### 총수입금액 귀속시기 (소득세법 §39)

| 거래 유형 | 귀속시기 |
|---|---|
| 재화 판매 (동산·부동산) | 인도일 (납품·소유권 이전일) |
| 용역 제공 | 용역 완료일 (계속 용역: 기간별 안분) |
| 임대소득 | 약정 지급일 (선수임대료: 받은 날) |
| 사용료·이자 | 지급받은 날 또는 약정일 |
| 부동산 양도 (사업 목적) | 잔금 수령일과 소유권이전등기일 중 **빠른 날** |

**실무 포인트**: 연말 미출고 매출(약정 완료, 미인도)은 인도 전이면 다음 연도 귀속. 대손금은 별도 처리.

### 비과세 사업소득 (소득세법 §12①)

| 항목 | 요건 |
|---|---|
| 논·밭을 직접 경작하여 발생한 소득 | 전액 비과세 |
| 1개 주택 임대소득 | 기준시가 12억원 이하, 국내주택 한정 |
| 농어가부업소득 | 연 3,000만원 이하 (조특법 §66) |
| 전통주 제조 소득 | 연 1,200만원 이하 |
| 조림기간 5년 이상 임지 임목 양도 | 연 600만원 이하 |

## Ch05. 근로소득

### 근로소득 수입시기 (소득세법 §49)

| 유형 | 수입시기 |
|---|---|
| 급여 | **근로를 제공한 날** (지급일 아님) |
| 잉여금 처분에 의한 상여 | 잉여금 처분 결의일 |
| 법인세법상 처분 상여 (소득처분) | 해당 법인 사업연도 종료일 |
| 임원 퇴직금 → 근로소득 재분류 | 퇴직일 |
| 주식매수선택권(스톡옵션) 행사이익 | 행사일 |

**실무 포인트**: 1월 귀속 급여를 2월에 지급해도 근로 제공월(1월)이 수입시기. 연말정산 귀속연도 결정에 중요.

## Ch14. 신고·납부

### 양도소득 예정신고 (소득세법 §105~§106)

**예정신고 의무**: 부동산·주식 양도 시 양도일 속하는 달의 말일부터 **2개월 이내** 신고·납부.

| 자산 종류 | 예정신고 기한 |
|---|---|
| 토지·건물 | 양도일 말일 + 2개월 |
| 부동산에 관한 권리 | 동일 |
| 비상장주식 | 동일 |
| 상장주식 (대주주) | 동일 |

**확정신고**: 예정신고 여부와 무관하게 다음 해 5월 종합소득세 확정신고 의무 (양도소득만 있어도 5월 확정신고).

**가산세**:
- 예정신고 미이행 시 무신고가산세 20% + 납부지연가산세 (1일 0.022%)
- 예정신고 이행 시 가산세 없음

**분할납부**: 납부세액 1,000만원 초과 시 신고기한 후 2개월 이내 분할납부 가능.

## Ch15. 비거주자

### 조세조약 제한세율

조세조약 체결국 거주자는 국내세법 세율보다 낮은 **제한세율** 적용 가능.

| 소득 유형 | 국내세법 | OECD 표준 | 미국 | 일본 | 중국 |
|---|---|---|---|---|---|
| 이자소득 | 14% | 10% | 12% | 10% | 10% |
| 배당소득 | 14% | 15% (5%¹) | 15% (10%¹) | 15% (5%¹) | 10% (5%¹) |
| 사용료(로열티) | 20% | 10% | 15% | 10% | 10% |
| 사업소득 | PE 있으면 과세 | PE 없으면 비과세 | — | — | — |

*¹ 일정 지분율(통상 10~25%) 이상 법인 수취 시 낮은 세율.*

**절차**: 제한세율 적용을 받으려면 **지급일 전까지** 원천징수의무자에게 「조세조약에 의한 비과세·면제 신청서」 제출 필요 (§156의2).

- 신청서 미제출 시 국내법 세율로 원천징수
- 사후 과다 원천징수 분은 경정청구로 환급 가능 (신청서 미제출분 포함)

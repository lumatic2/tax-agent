## 2025 Q51 — 금융소득 합산금액

**주제**: 의제배당·Gross-up 대상 판정 + 2천만 초과분 Gross-up 계산

**원본 정답**: ③ 40,350,000원 (가답안 PDF `2025년 제62회 세무사 1차 시험 가답안` 기준)

> ⚠️ 이 문제를 풀던 중 기존 정답 JSON이 **40문항 중 31건 파싱 오류**임이 드러남.
> 2023·2024·2025 3개 연도 세법학개론 정답 JSON을 원본 PDF(2023·2024는 OCR 이미지
> 직접 확인, 2025는 텍스트 추출) 기준으로 **전수 재작성 완료 (2026-04-12)**.
> 재작성 후 Q51 정답 = ③으로 정상 로드 확인.

### B 풀이 (LLM 단독)

**항목별 분류 (기억 기반)**
| # | 금액 | 해석 | 구분 |
|---|---|---|---|
| (1)a | 3,000,000 | 소비대차 전환분 이자 | 이자소득 |
| (1)b | 4,000,000 | 외상매출금 연장이자 | 사업소득 (제외) |
| (2) | 4,500,000 | ELB(파생결합사채) | 이자소득 |
| (3) | 3,500,000 | 주식발행초과금 자본전입 중 자기주식 미배정 지분율 상승분 | 배당 (비Gross-up) |
| (3)b | 4,500,000 | 나머지 자본준비금 | 과세 제외 |
| (4) | 10,000,000 | 자기주식처분이익 자본전입 | 배당 (Gross-up) |
| (5) | 17,500,000 | 인정배당 (결산확정일 2025.3.15 귀속) | 배당 (Gross-up) |

- 이자소득: 7,500,000 / 배당소득: 31,000,000 / 원금: 38,500,000
- Gross-up 대상 27,500,000, 비Gross-up 3,500,000
- 2천만 초과분 18,500,000 × Gross-up 비율 → 결과
- **B 예측: ③ 40,350,000** (단, Gross-up 비율 10%/11% 확신 없음 → 계산 검증 필요)

### A 풀이 (tax_calc_cli + law-mcp)

**도구 호출**
1. `mcp__law-mcp__search_law("소득세법")` → law_id=001565
2. `mcp__law-mcp__get_law_article(001565, "17")` → §17 배당소득 전문

**조문에서 확인된 사실**
- §17③ 본문: Gross-up 비율 = **100분의 10** (현행 10% 확정)
- §17③ 단서 제4호: §17②5 (자기주식 보유상태 자본전입 지분율 상승분) = **Gross-up 제외**
- §17③ 단서 제2호: §17②2가 **자기주식 소각이익** 자본전입 = Gross-up 제외
  - ⚠️ **"소각이익" ≠ "처분이익"**. 자기주식처분이익은 2011년부터 법인세 익금이라 Gross-up 대상에 포함
- §17①4: 법인세법상 배당 처분액 (인정배당) = §17③ 본문 적용 → Gross-up 대상

**계산**
- 2천만 초과분 18,500,000 × 10% = 1,850,000
- 금융소득금액 = 38,500,000 + 1,850,000 = **40,350,000**
- **A 예측: ③ 40,350,000** ✓

### 분석

| 구분 | B | A |
|---|---|---|
| 정답 여부 | ✓ (정답) | ✓ (정답) |
| 확신도 | 낮음 (Gross-up 10/11% 기억 불확실) | **높음** (조문 직접 확인) |
| 도구 기여 | — | 법조문 정밀도 ↑, 확신도 ↑ |

**교훈**
- 정답은 같아도 A는 근거 조항을 명시할 수 있음 → 설명·검증 가능성에서 우위
- Gross-up 비율(10%) 같은 숫자는 LLM 기억이 불안정 → **law-mcp 필수**
- `tax_calc_cli`는 이번 문제에선 호출 불필요 (금융소득 Gross-up 함수 없음 — 추후 추가 고려)

**발견된 인프라 이슈 (해결 완료)**
- 정답 JSON 파싱 오류 31/40 (세무사 1차 2025) → 3개 연도 전수 재작성 완료 (2026-04-12)
- 세무사 1차 2023·2024 정답 PDF는 이미지 기반 → pdfplumber `page.to_image()` + 멀티모달 Read로 직접 판독
- CPA 1차 2024·2025·2026 정답 JSON 전수 검증 결과 **모두 정상** (세무사 1차 파서만 국소 오류)
- CPA 1차 2026 PDF는 pdfplumber 파싱 실패 → **PyMuPDF(fitz)가 정상 동작** → 향후 PDF 파이프라인 대체 후보

## 2025 Q52 — 기타소득금액(ㄱ) · 원천징수세액(ㄴ)

**주제**: 6개 항목 기타소득 판별 + 필요경비 의제 + 직무발명 비과세 합산 한도 + 연금외수령 분류

**원본 정답**: ④ ㄱ 18,700,000원 / ㄴ 1,340,000원

### B 풀이 (LLM 단독)

내 기억 기반 계산은 ㄱ 21,700,000으로 선택지와 불일치. 직무발명 비과세 한도를 "근로·기타 각 1,000만원 별도"로 가정했다가 "합산 700만원"으로 보정하는 과정에서 경로 오류. 최종 **찍기 ④** (선택지 ㄴ 1,340,000이 (5) 제외 시만 성립한다는 역산으로 도달).

- **B 예측: ④** (확신도 낮음, 풀이 경로 불확실)

### A 풀이 (law-mcp)

**도구 호출**
1. `get_law_article(001565, "12")` → §12 비과세소득 전문 (§12③어·§12⑤라 = 직무발명보상금)
2. `get_law_article(001565, "21")` → §21 기타소득 전문 (§21①7·10·21·22의2·23)
3. `get_law_article(003956, "17의3")` → 근로 직무발명 비과세 "연 700만원"
4. `get_law_article(003956, "18")` → 기타 직무발명 비과세 "**연 700만원 중 근로 비과세 차감 잔액**" (합산 구조 확정)
5. `get_law_article(003956, "87")` → 필요경비 의제 80%(1호): 공익법인 시상·주택입주지체상금 / 60%(1의2): §21①7·8의2·9·15·19
6. `get_law_article(001565, "22")` → 퇴직소득 (이연퇴직 연금외수령 특례는 위임 조문으로 확인)

**결정적 조문**
- **시행령 §18②**: 기타소득 직무발명 비과세 = "연 700만원(근로 §12③어 비과세분 차감)" → 근로+기타 **합산 700만** 확정
- **시행령 §87**: 산업재산권 §21①7 = 60% 의제 / 주택입주지체상금 §21①10 단서 다목 = **80% 의제** (일반 위약금은 의제 없음)

**항목별 A 계산**
| # | 해석 | 기타소득금액 | 원천징수 |
|---|---|---|---|
| (1) 산업재산권 4M §21①7 | 60% 의제 → 40% | 1,600,000 | 320,000 |
| (2) 뇌물 30M, 몰수 20M | 판례상 몰수분 제외 | 10,000,000 | 0 (원천징수 대상 아님) |
| (3) 위약금 2.5M (계약금 대체 2M 포함) | 의제 없음 전액 과세 | 2,500,000 | 500K×20%=100,000 (계약금 대체분 §127 제외) |
| (4) 주택입주지체상금 8M §21①10 다목 | 80% 의제 → 20% | 1,600,000 | 320,000 |
| (5) 연금외수령 3M §21①21 | 이연퇴직 원천 → 퇴직소득으로 분리 (실무 해석) | 0 | 0 |
| (6) 직무발명 퇴직후 4M | 근로 600만 비과세 후 한도 잔여 100만 → 300만 과세, 의제 없음 | 3,000,000 | 600,000 |

- ㄱ = 1.6M + 10M + 2.5M + 1.6M + 0 + 3M = **18,700,000** ✓
- ㄴ = 320K + 0 + 100K + 320K + 0 + 600K = **1,340,000** ✓
- **A 예측: ④** ✓ (정확한 경로)

### 분석

| 구분 | B | A |
|---|---|---|
| 정답 여부 | ✓ (역산 찍기) | ✓ |
| 경로 정확도 | 낮음 — (5)/(6) 대칭 오해 | **높음** — 시행령 §18② 직접 확인 |
| 도구 기여 | — | **결정적**: 직무발명 합산 한도(시행령 §18②), 필요경비 의제 구분(시행령 §87) 모두 조문 인용 가능 |

**교훈**
- 직무발명 한도가 "근로·기타 별도" vs "합산"은 LLM 기억만으론 확정 불가 → **law-mcp 필수**
- 필요경비 의제 60% vs 80% 경계 (산업재산권 60%, 주택입주지체상금 80%)도 시행령 §87에서만 명시
- 연금외수령 3M 해석은 문제 자체에 모호성 있음 — 실무 해석(이연퇴직 원천→퇴직소득 분리)을 알아야 정답 도달
- 여전히 `tax_calc_cli`는 기타소득 계산 함수가 없어 미사용 → **2단계 작업: 기타소득 CLI 추가 필요**

## 2025 Q53 — 임원 퇴직소득 한도(ㄱ) · 퇴직소득금액(ㄴ)

**주제**: §22③ 임원 퇴직급여 한도 공식(B구간 ×3 / C구간 ×2) + 영§42의2⑥ "2011.12.31 이전분" 선택권 + 국민연금 일시금 한도 제외

**원본 정답**: ⑤ ㄱ 354,400,000원 / ㄴ 409,400,000원

**문제 요지**
- 甲 입사 2011.1.20 → 퇴직 2025.2.20 (전무이사, 임원)
- 지급: 퇴직금 450M + 퇴직위로금 14M + 국민연금 일시금 20M
- 2019.12.31 이전 3년 연평균 96M / 퇴직 전 3년 연평균 120M
- 2011.12.31 퇴직가정 퇴직급여 35M (임원퇴직급여지급규정)
- 선택 가능시 근로소득 적게 계산되는 방법 선택

### B 풀이 (LLM 단독)

기억 기반 계산. §22③ 임원 한도 공식을 구간별로 적용하고, 2011.12.31 이전분(35M)은 "한도 대상 기본금액에서 차감" 처리.
- B구간(2012~2019, 96개월) = 96M × 10% × 8 × 3 = **230.4M**
- C구간(2020.1.1~2025.2.20, 61개월 20일 → 62개월 절상) = 120M × 10% × 62/12 × 2 = **124M**
- 한도 ㄱ = 354.4M
- 한도 대상 기본 = 464M − 35M = 429M → 초과 74.6M 근로소득화
- 퇴직소득 ㄴ = 484M − 74.6M = 409.4M
- **B 예측: ⑤** (단, "2011.12.31 이전분을 A로 더하는지 vs 기본금액에서 빼는지" 구조가 기억상 흔들려 law-mcp 확정 필요)

### A 풀이 (law-mcp)

**도구 호출**
1. `get_law_article(001565, "22")` → 퇴직소득
2. `get_law_article(001565, "48")` → 퇴직소득공제 (근속연수 정의)
3. `get_law_article(003956, "42의2")` → 퇴직소득 범위 (임원 한도 상세)

**조문에서 확정된 사실**
- **§22③ 단서**: 임원 한도 비교 대상 퇴직소득금액에서 "①항1호(=공적연금)는 **제외**, 2011.12.31 가정 퇴직금이 있으면 그 금액을 **뺀 금액**"이 한도 초과 여부의 기준. → 국민연금 일시금 20M은 애초에 한도 밖, 2011.12.31 이전분은 기본금액에서 차감.
- **§22④1**: 한도 공식 근무기간 = 월수, 1월 미만 절상.
- **영§42의2⑥**: "2011.12.31 가정 퇴직금" = 두 가지 선택
  - ① 비율법: 퇴직소득금액 × (2011.12.31 이전 월수 ÷ 전체 월수)
  - ② 규정법: 2011.12.31 당시 임원퇴직급여지급규정에 따른 금액 (적용하기로 선택한 경우)
  - → 납세자 선택권 명문화.
- **§48**: 퇴직소득공제 근속연수(1년 미만 1년)는 §22③ 한도 공식과 무관 — §22④1이 독립 적용.

**계산**
- 전체 근무월수: 2011.1.20~2025.2.20 = 169개월 (정확히 14년 1개월, 일 단수 0)
- 2011.12.31 이전 월수: 2011.1.20~2011.12.31 = 11개월 12일 → 12개월 절상
- 2012.1.1~2019.12.31 = 96개월 (8년)
- 2020.1.1~2025.2.20 = 61개월 20일 → 62개월 절상

**A금액 선택 (문제(5))**
- 비율법 A = 464M × (12/169) ≈ 32.94M
- 규정법 A = 35M
- 35M 이 더 커서 한도 대상 기본금액이 더 작아짐 → 근로소득 최소화 → **규정법 35M 선택**

**한도 ㄱ**
- B = 96M × 1/10 × 96/12 × 3 = 230,400,000
- C = 120M × 1/10 × 62/12 × 2 = 124,000,000
- **ㄱ = 354,400,000** ✓

**퇴직소득금액 ㄴ**
- 한도 대상 기본 = 450M + 14M − 35M = 429,000,000 (국민연금 20M 제외)
- 초과분 = 429M − 354.4M = 74,600,000 → §22③에 따라 근로소득 재분류
- 퇴직소득금액 = (국민연금 20M) + (2011.12.31 이전분 35M) + (한도 내 354.4M) = **409,400,000** ✓
- (검산: 총 484M − 근로 재분류 74.6M = 409.4M)
- **A 예측: ⑤** ✓

### 분석

| 구분 | B | A |
|---|---|---|
| 정답 여부 | ✓ | ✓ |
| 확신도 | 중간 (구조 기억 흔들림) | **높음** (영§42의2⑥ 선택권 조문 직접 확인) |
| 도구 기여 | — | **결정적**: "2011.12.31 이전분" 비율법/규정법 선택권(영§42의2⑥)과 국민연금 20M 한도 제외(§22③ 단서) 모두 조문 명문화 확인 |

**교훈**
- 임원 한도 문제의 핵심은 공식(×3/×2) 자체가 아니라 **기본금액 산정에서 무엇을 빼는지** — 영§42의2⑥의 선택권이 정답을 가르는 포인트. 기억만으론 A금액을 "한도에 더한다"로 착각하기 쉽다.

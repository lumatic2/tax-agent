# Tax Agent SKILL — 세무 지식 참조

소득세법 각 챕터에서 Python 계산 함수 없이 텍스트 안내가 필요한 항목.
사용자 질문에 답변할 때 이 내용을 참조한다.

---

## Ch01. 소득세 총설

### 납세지 규정 (소득세법 §6~§8)

납세지 = 세무신고·납부 관할 세무서를 결정하는 기준지.

| 납세자 유형 | 납세지 원칙 |
|---|---|
| 거주자 | 주소지 (주소 없으면 거소지) |
| 비거주자 | 국내원천소득 발생지 (부동산: 소재지, 사업: 사업장) |
| 사업소득자 — 사업장 선택 시 | 사업장 소재지 (§8 납세지신고서 제출 필요) |

- **납세지 변경**: 주소 이전 시 자동 변경, 별도 신고 불요
- **사업장 납세지신고**: 사업소득자가 주소지 대신 사업장을 납세지로 원하면 사업장 관할 세무서에 신고서 제출 (§8①)
- **공동사업장**: 대표 공동사업자 주소지 또는 사업장 소재지

## Ch02. 이자소득

### 이자소득 수입시기 (소득세법 §45①1)

수입시기 = 이자소득이 귀속되는 과세기간 결정 기준.

| 이자 유형 | 수입시기 |
|---|---|
| 약정에 의해 지급일이 정해진 이자 | **약정일** |
| 약정일 이전에 실제 지급된 이자 | **지급일** (앞당겨진 날) |
| 채권·증권 만기 이자 | 만기일 (중도상환 시 상환일) |
| 보통예금·수시입출금 이자 | 실제 통장 입금일 |
| 원본에 전입되는 이자 | 원본 전입일 |

**실무 포인트**: 12월 만기 정기예금 이자를 1월에 수령해도 → 만기일(12월)이 수입시기. 해당 과세연도 귀속.

## Ch03. 배당소득

### 집합투자기구 배당소득 특례 (소득세법 §17①5)

집합투자기구(펀드·ETF 등)에서 받는 이익분배금의 과세 처리.

| 구분 | 과세 여부 |
|---|---|
| 환매(해지) 시 수익: 환매금액 - 취득가액 | 배당소득 과세 (원천징수 14%) |
| 펀드 내 국내주식 매매익 | **비과세** (§94①3 양도소득 과세 대상 아님) |
| 펀드 내 채권 이자 | 이자소득으로 구분 과세 |
| 분배금(결산 지급) | 배당소득 과세 |
| 해외주식형 ETF(국내 상장) — 주식매매익 분배 | 배당소득 과세 (해외주식 양도익은 비과세 아님) |

- **Gross-up 불적용**: 집합투자기구 배당소득에는 배당세액공제(Gross-up) 적용 안 됨 (§17③ 단서)
- **손실 환매**: 환매금액 < 취득가액이면 배당소득 0원 (음수 불인정)

## Ch04. 사업소득

### 총수입금액 귀속시기 (소득세법 §39)

| 거래 유형 | 귀속시기 |
|---|---|
| 재화 판매 (동산·부동산) | 인도일 (납품·소유권 이전일) |
| 용역 제공 | 용역 완료일 (계속 용역: 기간별 안분) |
| 임대소득 | 약정 지급일 (선수임대료: 받은 날) |
| 사용료·이자 | 지급받은 날 또는 약정일 |
| 부동산 양도 (사업 목적) | 잔금 수령일과 소유권이전등기일 중 **빠른 날** |

**실무 포인트**: 연말 미출고 매출(약정 완료, 미인도)은 인도 전이면 다음 연도 귀속. 대손금은 별도 처리.

### 비과세 사업소득 (소득세법 §12①)

| 항목 | 요건 |
|---|---|
| 논·밭을 직접 경작하여 발생한 소득 | 전액 비과세 |
| 1개 주택 임대소득 | 기준시가 12억원 이하, 국내주택 한정 |
| 농어가부업소득 | 연 3,000만원 이하 (조특법 §66) |
| 전통주 제조 소득 | 연 1,200만원 이하 |
| 조림기간 5년 이상 임지 임목 양도 | 연 600만원 이하 |

## Ch05. 근로소득

### 근로소득 수입시기 (소득세법 §49)

| 유형 | 수입시기 |
|---|---|
| 급여 | **근로를 제공한 날** (지급일 아님) |
| 잉여금 처분에 의한 상여 | 잉여금 처분 결의일 |
| 법인세법상 처분 상여 (소득처분) | 해당 법인 사업연도 종료일 |
| 임원 퇴직금 → 근로소득 재분류 | 퇴직일 |
| 주식매수선택권(스톡옵션) 행사이익 | 행사일 |

**실무 포인트**: 1월 귀속 급여를 2월에 지급해도 근로 제공월(1월)이 수입시기. 연말정산 귀속연도 결정에 중요.

## Ch14. 신고·납부

### 양도소득 예정신고 (소득세법 §105~§106)

**예정신고 의무**: 부동산·주식 양도 시 양도일 속하는 달의 말일부터 **2개월 이내** 신고·납부.

| 자산 종류 | 예정신고 기한 |
|---|---|
| 토지·건물 | 양도일 말일 + 2개월 |
| 부동산에 관한 권리 | 동일 |
| 비상장주식 | 동일 |
| 상장주식 (대주주) | 동일 |

**확정신고**: 예정신고 여부와 무관하게 다음 해 5월 종합소득세 확정신고 의무 (양도소득만 있어도 5월 확정신고).

**가산세**:
- 예정신고 미이행 시 무신고가산세 20% + 납부지연가산세 (1일 0.022%)
- 예정신고 이행 시 가산세 없음

**분할납부**: 납부세액 1,000만원 초과 시 신고기한 후 2개월 이내 분할납부 가능.

## Ch15. 비거주자

### 조세조약 제한세율

조세조약 체결국 거주자는 국내세법 세율보다 낮은 **제한세율** 적용 가능.

| 소득 유형 | 국내세법 | OECD 표준 | 미국 | 일본 | 중국 |
|---|---|---|---|---|---|
| 이자소득 | 14% | 10% | 12% | 10% | 10% |
| 배당소득 | 14% | 15% (5%¹) | 15% (10%¹) | 15% (5%¹) | 10% (5%¹) |
| 사용료(로열티) | 20% | 10% | 15% | 10% | 10% |
| 사업소득 | PE 있으면 과세 | PE 없으면 비과세 | — | — | — |

*¹ 일정 지분율(통상 10~25%) 이상 법인 수취 시 낮은 세율.*

**절차**: 제한세율 적용을 받으려면 **지급일 전까지** 원천징수의무자에게 「조세조약에 의한 비과세·면제 신청서」 제출 필요 (§156의2).

- 신청서 미제출 시 국내법 세율로 원천징수
- 사후 과다 원천징수 분은 경정청구로 환급 가능 (신청서 미제출분 포함)

## Ch01. 부가가치세 총설

### 사업자등록 (부가가치세법 §8)

사업을 시작하는 자는 사업 개시일로부터 **20일 이내**에 사업장 관할 세무서에 등록 신청.

| 구분 | 내용 |
|---|---|
| 등록 기한 | 사업 개시일부터 20일 이내 |
| 신규 사업자 | 사업 개시 전 등록 가능 (사전등록) |
| 미등록 가산세 | 공급가액 × 1% (§60①: 사업 개시일 ~ 등록 신청일 직전일 공급가액) |
| 등록 전 매입세액 | 등록 신청일부터 역산 20일 이내 매입분은 공제 허용 (§39①7 단서) |
| 간이과세자 | 직전연도 공급대가 8,000만 미만이면 간이과세로 자동 등록 |

**실무 포인트**: 사업자등록 전 매입세액은 원칙적으로 불공제이나, 등록 신청일 전 20일 이내분은 예외적으로 공제. 개업 준비 중 설비·인테리어 매입이 많은 경우 가능한 빨리 사전등록 권장.

### 사업장 판정 (부가가치세법 §6)

사업장 = 사업자가 사업을 하는 장소. 부가세 신고·납부 단위.

| 사업 유형 | 사업장 판정 |
|---|---|
| 고정된 장소에서 사업 | 그 장소 |
| 고정된 장소 없이 사업 (행상 등) | 사업자 주소 또는 거소 |
| 건설업·운수업 등 | 사업 허가 관청 소재지 |
| 비거주자·외국법인 | 국내 사업장 (영업소·사무소) |
| 사업장이 2개 이상인 경우 | **각 사업장별** 별도 신고 (§4: 총괄납부 특례 가능) |

- **총괄납부**: 2개 이상 사업장의 납부세액을 주된 사업장에서 총괄 납부 가능 (§4, 관할 세무서장 승인 필요)
- **사업자단위과세**: 사업장마다 별도 사업자등록 대신 사업자단위로 하나의 등록번호 사용 가능 (§8⑧, 신청 필요)

## Ch02. 과세거래

### 재화의 수입 (부가가치세법 §12)

외국으로부터 우리나라에 반입하는 재화(보세구역 경유 시 보세구역에서 반출).

| 구분 | 내용 |
|---|---|
| 납세의무자 | 수입하는 자 (사업자 여부 불문) |
| 과세표준 | 관세 과세가격 + 관세 + 개별소비세 + 주세 + 교육세 + 농특세 (§29②) |
| 세율 | 10% |
| 납부 | 수입 시 세관장에게 납부 (통관 시) |
| 매입세액 공제 | 사업자가 과세사업용으로 수입 시 수입세금계산서로 공제 가능 |

**영세율 vs 수입**: 수출은 영세율(0%), 수입은 10% 과세. 보세구역 내 거래는 수입 아님(반출 시점에 과세).

## Ch03. 영세율과 면세

### 면세 포기 (부가가치세법 §27)

면세사업자가 과세사업자로 전환하려는 경우 면세 포기 신고 가능.

| 항목 | 내용 |
|---|---|
| 신고 기한 | 적용받으려는 과세기간 개시일 **20일 전**까지 |
| 효과 | 면세 → 과세로 전환, 매입세액 공제 가능 |
| 최소 적용 | 3년간 계속 적용 (중도 철회 불가) |
| 포기 가능 대상 | 제한 없음 (모든 면세 재화·용역) |
| 실익이 있는 경우 | 수출업자(영세율 적용 시 매입세액 환급), 면세 매입비용 큰 사업자 |

**실무 포인트**: 면세 포기 실익 판단 — 매입세액 환급 > 매출세액 납부 이면 유리. 특히 영세율 적용 가능한 수출 면세업자가 대표적.

### 조특법 영세율 특례 (조세특례제한법 §105~§105의2)

| 대상 | 근거 | 내용 |
|---|---|---|
| 외교관 면세 물품 | §105①1 | 외국 공관·외교관에 공급하는 재화·용역 |
| 주한미군 공급 | §105①2 | 주한미군 또는 그 구성원에 공급 |
| 국제기구 공급 | §105①4 | 국제기구에 공급하는 재화·용역 |
| 내국신용장·구매확인서 | §105①5 | 영세율 적용 (수출용 원재료 국내 공급) |
| 장애인용 보장구 | §105①6 | 장애인 보조기구 제조·수입 |

## Ch04. 세금계산서

### 수정세금계산서 발급 사유 (부가가치세법 §32②)

이미 발급한 세금계산서의 기재사항에 착오·정정이 필요한 경우.

| 발급 사유 | 발급 시기 |
|---|---|
| 환입(반품) | 환입된 날 |
| 계약 해제 | 계약 해제일 |
| 공급가액 추가·차감 | 추가·차감 사유 발생일 |
| 내국신용장 등 사후 개설 | 개설일 |
| 착오로 이중 발급 | 착오 확인일 |
| 착오 기재 (필요적 기재사항) | 확인 즉시 |

**실무 포인트**: 수정세금계산서는 당초 세금계산서와 함께 보관. 공급받는 자도 수정분 반영하여 매입세액 가감 필요.

### 전자세금계산서 (부가가치세법 §32②, 영§68)

| 구분 | 의무 발급 기준 |
|---|---|
| 법인사업자 | **전면 의무** (규모 무관) |
| 개인사업자 | 직전연도 공급가액(면세 포함) **1억원 이상** |
| 발급 기한 | 공급시기가 속하는 달의 다음 달 **10일**까지 전송 |
| 전자발급 세액공제 | 건당 200원, 연 100만원 한도 (§47: 개인사업자만) |
| 미전송 가산세 | 공급가액 × 0.5% (§60: 기한 후 전송은 0.3%) |

### 영수증 발급 대상 (부가가치세법 §36)

세금계산서 대신 영수증을 발급하는 업종:

| 업종 | 비고 |
|---|---|
| 소매업 | 최종 소비자 대상 |
| 음식·숙박업 | |
| 여객운송업 | |
| 입장권 판매업 | |
| 미용·목욕·세탁업 | |
| 의료업 | 면세와 구분 주의 |
| 기타 대통령령 업종 | 영§73 |

- 사업자가 요청하면 영수증 대신 **세금계산서 발급 의무** (§36① 단서)
- 영수증 발급 사업자도 세금계산서 발급은 가능 (거래상대방 요청 시)

## Ch08. 신고와 납부

### 예정신고·확정신고 기한 (부가가치세법 §48~§49)

| 구분 | 대상기간 | 신고기한 | 비고 |
|---|---|---|---|
| 1기 예정 | 1.1~3.31 | **4.25** | 개인: 예정고지(직전기 50%) |
| 1기 확정 | 4.1~6.30 (또는 1.1~6.30) | **7.25** | |
| 2기 예정 | 7.1~9.30 | **10.25** | 개인: 예정고지 |
| 2기 확정 | 10.1~12.31 (또는 7.1~12.31) | **1.25** | |

- **개인사업자 예정신고**: 원칙적으로 예정고지(직전 과세기간 납부세액 50%)로 갈음. 단, 휴업·사업부진·조기환급 사유 있으면 직접 예정신고 가능.
- **법인사업자**: 예정·확정 모두 직접 신고

### 환급 유형 (부가가치세법 §59)

| 구분 | 일반환급 | 조기환급 |
|---|---|---|
| 환급 시기 | 확정신고 기한 후 **30일 이내** | 신고 기한 후 **15일 이내** |
| 대상 | 모든 환급 | 영세율 적용, 시설투자 관련 |
| 신청 | 별도 불요 (확정신고에 포함) | 조기환급 신고서 제출 |

**실무 포인트**: 수출업자(영세율)는 매출세액 0 + 매입세액 공제 = 환급. 조기환급 신청하면 15일 이내 환급받을 수 있어 자금 운용에 유리.

### 대리납부 (부가가치세법 §52)

국외사업자가 국내에 용역 공급 시, 공급받는 자가 부가세를 대리납부.

| 항목 | 내용 |
|---|---|
| 대상 | 국외사업자로부터 용역 공급받는 자 |
| 세율 | 10% |
| 신고 기한 | 대가 지급일이 속하는 달의 다음 달 25일 |
| 매입세액 공제 | 과세사업자가 과세사업용이면 공제 가능 |

**실무 포인트**: 해외 SaaS·클라우드 서비스 이용 시 대리납부 의무 발생. 대가 지급 시 10% 별도 납부.

# 물음 2 — 을의 근로소득 및 종합소득공제 풀이

---

## 요구사항 1 : 근로소득 총급여액 · 공제액 · 소득금액

### STEP 1. ㈜A 과세 근로소득 항목 분류

| 항목 | 원금액 | 비과세 여부 | 총급여 산입액 |
|---|---|---|---|
| 기본급 (사용자 대리납부 500,000원 포함) | 20,000,000 | 과세 (대리납부액은 근로소득으로 과세) | **20,000,000** |
| 자녀학자보조금 | 1,000,000 | 비과세 학자금은 **본인** 수업료에 한함 → 과세 | **1,000,000** |
| 식대 (월 300,000 × 10개월) | 3,000,000 | 식사 미제공 시 비과세 한도 **월 200,000원** → 비과세 200,000×10 = 2,000,000 | **1,000,000** |
| 퇴직급여 | 30,000,000 | **퇴직소득** — 근로소득 아님 | **0** |
| **소계** | | | **22,000,000** |

### STEP 2. ㈜B 일용근로소득 처리

일용근로자 급여(400,000원×40일 = 16,000,000원) 및 성과급(7,000,000원)은 **일용근로소득으로 분리과세** → 종합소득에 합산하지 않음.

> **근로소득 총급여액 = 22,000,000원**

### STEP 3. 근로소득공제 (공제표 — 1,500만원 초과 4,500만원 이하 구간)

```
7,500,000 + (22,000,000 − 15,000,000) × 15%
= 7,500,000 + 7,000,000 × 0.15
= 7,500,000 + 1,050,000
= 8,550,000원
```

### STEP 4. 근로소득금액

```
22,000,000 − 8,550,000 = 13,450,000원
```

## 요구사항 2 : 인적공제 및 특별소득공제

### STEP 5. 기본공제 대상자 판단 (연간 소득금액 100만원 이하 & 나이 요건)

| 대상자 | 나이 | 나이 요건 | 소득금액 | 소득 요건 | 기본공제 |
|---|---|---|---|---|---|
| 을 (본인) | 45세 | 무관 | — | 무관 | ✅ |
| 배우자 | 48세 | 무관 | 사업소득 1,000,000 | ≤ 100만원 ✅ | ✅ |
| 부친 | 74세 | ≥60세 ✅ | 양도소득 500,000 | ≤ 100만원 ✅ | ✅ |
| 아들 | 16세 | ≤20세 ✅ | 없음 | ✅ | ✅ |

**기본공제 인원 : 4명**
```
4명 × 1,500,000 = 6,000,000원
```

### STEP 6. 추가공제

| 항목 | 대상 | 금액 | 근거 |
|---|---|---|---|
| 경로우대공제 | 부친 (74세 ≥ 70세) | 1,000,000 | 소득세법 §51③1호 |
| 장애인공제 | 부친 (장애인) | 2,000,000 | 소득세법 §51③2호 |
| 부녀자공제 | 을 (배우자 있는 여성, 근로소득금액 13,450,000 ≤ 3,000만원) | 500,000 | 소득세법 §51①5호 |
| **합계** | | **3,500,000** | |

### STEP 7. 특별소득공제 (소득세법 §52)

**① 건강보험료·고용보험료 공제**
- 을이 부담할 건강보험료·고용보험료 500,000원 (사용자 대리납부 → 근로자 납부로 인정)
- 전액 공제 → **500,000원**

**② 주택임차차입금 원리금상환액 공제**
- 요건 : 무주택 세대주 ✅ / 국민주택 임차 ✅ / 금융기관 차입 ✅
- 공제율 40%, 연간 한도 3,000,000원
```
15,000,000 × 40% = 6,000,000 → 한도 적용 → 3,000,000원
```

**특별소득공제 합계**
```
500,000 + 3,000,000 = 3,500,000원
```
